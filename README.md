# AI Applications Portfolio – Cesar Noriega

## Overview
This portfolio presents an AI application focused on HVAC predictive maintenance in cruise ships.

## Project
### HVAC Predictive Maintenance
This project uses basic AI and machine learning concepts to analyze HVAC system data such as temperature, vibration, and humidity.

The goal is to detect abnormal conditions and identify potential failures before they occur.

## Tools Used
- Python
- Google Colab
- GitHub

## Application
This solution can be applied to:
- Cruise ships HVAC systems
- Industrial operations
